**dictionary** is a Meteor site that contains sample CRUD applets, using the dataset-* meteorite packages.

------------------------
### Installation

````sh
# Should be as simple as cloning the repository...  
git clone https://github.com/awatson1978/dictionary.git
cd dictionary

# And then running it...
meteor
````
